## 🛠️ Déploiement Rapide sur Render

<div align="center">

| Étape | Action |
|:---:|:---|
| **1** | Fork ce dépôt sur votre compte GitHub |
| **2** | Allez sur [render.com](https://render.com) et créez un nouveau Web Service |
| **3** | Connectez votre dépôt GitHub |
| **4** | Commande de démarrage : `npm start` |
| **5** | Visitez votre URL Render pour le panel de parrainage |
| **6** | Entrez votre numéro WhatsApp → Obtenez le code → Connectez ! |
